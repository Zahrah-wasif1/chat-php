üìã DEPLOYMENT STEPS:

1. UPLOAD FILES:
   - Upload entire 'server' folder to Cloudways public_html
   - Upload 'sock' folder outside public_html (for security)

2. SET DOCUMENT ROOT:
   - In Cloudways panel: Application Settings ‚Üí Document Root
   - Set to: public_html/public

3. CREATE .ENV FILE:
   - Location: public_html/.env (same level as public folder)
   - Copy from env.example and update with Cloudways database credentials
   - Get credentials from: Cloudways Panel ‚Üí Access Details

4. INSTALL DEPENDENCIES:
   SSH commands:
   cd /home/master/applications/[app-id]/public_html
   composer install --no-dev

5. SETUP SOCKET SERVER:
   SSH commands:
   cd /path/to/sock
   npm install
   Create .env file with API_URL and SOCKET_PORT
   pm2 start server.js --name chat-socket
   pm2 save
   pm2 startup

6. DATABASE:
   - Import your database schema
   - Verify connection in .env file

7. PERMISSIONS:
   chmod -R 755 public_html/public

8. SSL:
   - Enable free SSL in Cloudways panel
